## Опис

Програма знаходить найменше власне значення матриці **A** степеневим методом:

```
A = [[2.3, 1.4, 0.6],
     [1.4, 1.7, 0.5],
     [0.6, 0.5, 1.3]]
```

## Метод

Степеневій метод знаходить найбільше за модулем власне значення. Для знаходження найменшого власного значення використовується степеневий метод до оберненої матриці **A⁻¹**:

- Найбільше власне значення λ_max(A⁻¹) відповідає найменшому власному значенню λ_min(A) = 1/λ_max(A⁻¹)
- На кожній ітерації розв'язується система **A·y = x** для обчислення **A⁻¹·x**
- Система розв'язується методом Гаусса з частковим вибором головного елемента
- Вектор нормалізується після кожної ітерації
- Процес продовжується до досягнення заданої точності

## Вихідні дані

Програма виводить:

1. **Таблицю ітерацій** степеневого методу з наближеннями найменшого власного значення
2. **Знайдене найменше власне значення** та відповідний власний вектор
3. **Порівняння з точним розв'язком** через `numpy.linalg.eig`
4. **Абсолютну та відносну похибки**
5. **Перевірку** рівності **A·v = λ·v** для знайденого розв'язку

## Алгоритм

1. Ініціалізація початкового вектора **x₀** (нормалізованого)
2. На кожній ітерації:
   - Розв'язування системи **A·y = x** методом Гаусса
   - Нормалізація вектора **y**
   - Обчислення наближення власного значення **A⁻¹** через скалярний добуток
   - Обчислення найменшого власного значення **A**: λ_min = 1/λ_max(A⁻¹)
   - Перевірка умови збіжності
3. Повернення результату після досягнення заданої точності
