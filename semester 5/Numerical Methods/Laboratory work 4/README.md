## Методи

### Пряма інтерполяція

1. На інтервалі [a, b] будується 10 рівновіддалених вузлів xᵢ
2. Обчислюються значення функції yᵢ = f(xᵢ) у вузлах
3. Побудова полінома Лагранжа P(x) ≈ f(x):
   ```
   P(x) = Σ yᵢ · Lᵢ(x)
   ```
   де Lᵢ(x) - базисні поліноми Лагранжа
4. Знаходження нулів полінома P(x) = 0 на інтервалі
5. Вибір найбільшого кореня

### Обернена інтерполяція

1. Вибираються вузли навколо y = 0 (де f(x) близька до нуля)
2. Побудова полінома Лагранжа Q(y) ≈ x (інтерполює x як функцію від y):
   ```
   Q(y) = Σ xᵢ · Lᵢ(y)
   ```
   де вузли - це пари (yᵢ, xᵢ), де yᵢ = f(xᵢ)
3. Знаходження кореня через Q(0)

## Поліном Лагранжа

Поліном Лагранжа ступеня n-1, що проходить через n точок (xᵢ, yᵢ), має вигляд:

```
P(x) = Σᵢ₌₀ⁿ⁻¹ yᵢ · Lᵢ(x)
```

де базисні поліноми Лагранжа:

```
Lᵢ(x) = ∏ⱼ≠ᵢ (x - xⱼ) / (xᵢ - xⱼ)
```

2. **Пряма інтерполяція**:
   - Побудова 10 рівновіддалених вузлів на інтервалі
   - Обчислення значень f(x) у вузлах

3. **Обернена інтерполяція**:
   - Вибір вузлів навколо y = 0
   - Побудова полінома Лагранжа Q(y) для x як функції від y

4. **Порівняння та візуалізація**: Порівняння результатів обох методів та побудова графіків
