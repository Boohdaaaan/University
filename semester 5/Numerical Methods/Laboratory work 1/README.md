# Лабораторна робота 1: Знаходження найменшого кореня нелінійного рівняння

**Автор:** Кузнецов Богдан
**Група:** ТК-31

## Опис

Програма знаходить найменший корінь рівняння **x³ - 3x² - 14x - 8 = 0** двома методами:

1. **Метод релаксації** (ітераційний метод)
2. **Модифікований метод Ньютона**

## Вимоги

- Python >= 3.10
- uv (менеджер пакетів)

## Встановлення

```bash
cd "Laboratory work 1"
uv sync
```

## Запуск

```bash
uv run python main.py
```

Програма дозволяє ввести власну точність або використати значення за замовчуванням (ε = 10⁻⁴).

## Вихідні дані

Програма виводить:

1. **Визначення інтервалу** для найменшого кореня
2. **Вибір форми методу релаксації** з перевіркою умов збіжності
3. **Апріорну оцінку** кількості ітерацій для методу релаксації
4. **Таблицю ітерацій** для методу релаксації
5. **Таблицю ітерацій** для модифікованого методу Ньютона
6. **Порівняння результатів** обох методів
7. **Графік функції** з позначенням знайденого кореня (зберігається у `graph.png`)

### Метод релаксації

Використовує форму x = x - λ·f(x), де параметр λ обирається для забезпечення збіжності до найменшого кореня.

### Модифікований метод Ньютона

Використовує формулу xₙ₊₁ = xₙ - f(xₙ)/f'(x₀), де похідна фіксується на початковому наближенні x₀.
